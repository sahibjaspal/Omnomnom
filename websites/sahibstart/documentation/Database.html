OMNOMNOM Database

The following tables were used created in the database:

  1. UserData - To store UserNames and passwords
  2. CommentTable - To store user search keys, outlet's YelpID and User Comment
  3. LikeStatus - To store user search keys, outlet's YelpID and User Like Status
  4. FollowStatus - To store usernames of users followed and User Follow Status
