Project Overview

Hi!

OMNOMNOM is a food related Social Networking website. This website has been built for food enthusiasts
to Like, Comment and Follow useractivities.

OMNOMNOM enables users to search for particular food outlets in a locality.
The users can view the various outlets that provide the food searched.
Users can select the an outlet to view more details, which displays the rating, reviews from yelp users,
Contact and Address details. Users can also Like restaurants. Reviews can also be added for each user.

Here is a brief description of the Pages involved in the Project.

  1. HomePage - The Home Page provides a search bar to search for Food.
  2. LoginPage - This page allows a user to Login into OMNOMNOM.
  3. SearchPage - This Page provides the search results and the details view of each outlet. Users can also Like outlets and Comment on each one of them.
  4. UserProfile - This is the profile page of each user which displays the following :
    • The User's Likes
    • The User's Comments
    • Following User's Likes
    • Following User's Comments
  5. OtherUserProfile - This is a profile page of Other Users which displays their Likes and Comments. Logged in Users can view other profiles and follow them.
